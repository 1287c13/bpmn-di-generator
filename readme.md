## Как пользоваться локально

- создаем BPMN диаграму во внешнем редакторе (например, в https://bpmn.io/), 
получаем схему, например такую:

<img alt="Исходная диаграма" src="examples/1/1.PNG">

- выгружаем схему в .bpmn файл средствами внешнего редактора 
- убеждаемся что у вас локально установлен Python (тестировалось на Python 3.11, 
предположительно требуется Python 3.8+)
- скачиваем скрипт ```bpmn-di-generator.py```
- запускаем скрипт -> открывается окно редактора
- нажимаем в редакторе кнопку "Выбрать файл" и выбираем скачанный .bpmn файл
- после того как содержимое файла отобразилось в окне, последовательно нажимаем
кнопки "Пересчитать di слой" и "Сохранить файл"
- В каталоге с исходным файлом появился файл с таким же именем и постфиксом 
"-autolayout", этот файл содержит обновленную диаграму
- загружаем обновленную диаграму обратно во внешний BPMN редактор
- раскладка BPMN схемы обновлена

## Ограничения

Часть ограничений будет снята по мере развития проекта, а пока:

- каждый процесс или субпроцесс должен начинаться со стартового события
  (или нескольких стартовых событий)
- развернутый субпроцесс должен происходить в рамках одной роли
- не поддерживается рендеринг артефактов
- не поддерживаются граничные события
- на схеме должен быть один процесс
- размеры элементов не пересчитываются
- координаты подписей к шлюзам и стрелкам не пересчитываются (сохраняется 
отступ, заданный в оригинальной раскладке)
- дорожки результирующей схемы расположены горизонтально
